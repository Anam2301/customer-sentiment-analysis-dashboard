📊 Customer Sentiment Analysis (Python + Power BI)

This project demonstrates an end-to-end sentiment analysis workflow using Python and Power BI.

---

🚀 Project Workflow

📥 Data Collection
- Customer review dataset

 🧹 Data Cleaning (Python)
- Removed null values
- Cleaned text (lowercase, trimming)
- Removed duplicates

 🤖 Sentiment Analysis (Python)
- Used NLP techniques to classify reviews
- Generated predicted sentiment

 🔄 Data Transformation
- Created final dataset with:
  - Review
  - Actual Sentiment
  - Predicted Sentiment
  - Result (Correct/Wrong)

 📊 Dashboard (Power BI)
- Built interactive dashboard to visualize insights

---

 📌 Key Features

- KPI Metrics:
  - Total Reviews
  - Positive Reviews
  - Negative Reviews
  - Model Accuracy

- Visualizations:
  - Sentiment Distribution
  - Actual vs Predicted Sentiment
  - Model Prediction Performance

- Interactive Filtering

---

 📊 Insights

- Positive sentiment slightly dominates (~50%)
- Model achieved **86% accuracy**
